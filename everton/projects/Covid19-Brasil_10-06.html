1- Introduo

Este projeto tem como objetivo fazer uma anlise dos dados sobre a pandemia da Covid-19 no Brasil, sobretudo em algumas cidades brasileiras de interesse. Os dados utilizados neste notebook so disponibilizados no Kaggle e podem ser acessados no seguinte link: Coronavirus - Brazil . Aps esta breve anlise, o objetivo construir um modelo de regresso capaz de prever o nmero de mortes causadas pelo vrus em datas futuras.

O arquivo de referncia para este estudo, "brazil_covid19_cities.csv", apresenta os seguintes dados sobre a pandemia no pas:

  • date: data de confirmao dos casos;
  • state: estado brasileiro de referncia dos casos;
  • name: nome da cidade de referncia dos casos;
  • code: cdigo da cidade;
  • cases: nmero de casos confirmados;
  • deaths: nmero de mortes confirmadas pelo Covid19.

    • Este dataset ser analisado juntamente com o dataset "brazil_population_2019.csv", tambm disponibilizado na pgina indicada anteriormente, para que se possa avaliar a situao dos casos proporcionalmente populao da cidade de referncia.

    # Criando uma funo que gera um plot que apresenta a evoluo do nmero de mortes dia a dia:
def deaths_per_day(city_name):
    """Cria um grfico apresentando as mortes dia a dia.
    
    Args:
    city_name: string indicando a cidade a ser filtrada. A conveno no dataset  'Cidade - UF'.
    """
    filt_city = covid_cap_br[covid_cap_br['city'] == city_name]

    filt_city = filt_city.reset_index(drop = True)

    filt_city['deaths_per_day'] = ''
    for i in range(0, len(filt_city.index)):
        if i == 0:
            filt_city['deaths_per_day'][i] = filt_city['deaths'][i]
        else:
            if (filt_city['deaths'][(i-1)]) <= (filt_city['deaths'][i]):
                filt_city['deaths_per_day'][i] = filt_city['deaths'][i] - filt_city['deaths'][(i-1)]
            else:
                filt_city['deaths_per_day'][i] = 0

    fig, ax = plt.subplots(nrows = 1, ncols = 1, figsize = (16, 5))

    sns.barplot(x = 'day_count', y = 'deaths_per_day', data = filt_city, palette = "autumn_r").set(
        xlabel = 'Contagem dos dias desde ' + min_date,
        ylabel = 'Mortes dirias',
        title = 'Evoluo diria do nmero de mortes em ' + city_name)
    fig.show()

    # Criando funo para desenvolver, treinar e avaliar o modelo:
def covid_poly_model(city_data, n_degree, seed):
    """Transforma os dados para uma regresso polinomial, cria o modelo de regresso linear e o avalia, juntamente com a
    apresentao de um grfico que ilustra os dados previstos pelo modelo em relao aos dados originais.
    
    Args:
    city_data: dataset filtrado pela funo city_filter();
    n_degree: nmero que representa o grau da transformao linear a ser aplicada;
    seed: nmero que permite a reproduo da separao dos dados de treino e de teste para posterior comparao.
    
    Returns:
    poly_regr: modelo de transformao polinomial a ser aplicado posteriormente, permitindo ao modelo fazer previses;
    model: modelo de regresso linear para fazer as previses.
    """
    # Separando as variveis:
    X = city_data[['day_count']]
    Y = city_data[['deaths']]

    # Aplicando a transformao polinomial aos dados:
    poly_regr = PolynomialFeatures(degree = n_degree)
    X_poly = poly_regr.fit_transform(X)

    # Criando dados de treinos e de teste:
    X_poly_train, X_poly_test, Y_train, Y_test = train_test_split(X_poly, Y,
                                                                 test_size = 0.2,
                                                                 random_state = seed)

    # Criando o modelo:
    model = LinearRegression()

    # Treinando o modelo:
    model.fit(X_poly_train, Y_train)

    # Fazendo as previses para os dados de teste:
    Y_pred = model.predict(X_poly_test)

    # Avaliando o modelo pela mtrica MAE (mean absolut error):
    mae = round(mean_absolute_error(Y_test, Y_pred))

    sns.scatterplot(x = 'day_count', y = 'deaths', data = city_data).set(
        xlabel = 'Contagem dos dias a partir de ' + min_date,
        ylabel = 'Nmero de mortos',
        title = 'Evoluo Mortes x Regresso Polinomial de Grau ' + str(n_degree))
    plt.plot(X, model.predict(X_poly), color = 'r')
    plt.show()

    print('O erro mdio absoluto da previso do nmero de mortos para a regresso polinomial de grau ' +
          str(n_degree) + '  de: ' + str(mae) + '.')

    return poly_regr, model
